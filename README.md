# Automobile Sales Analysis 🚗

## Overview
In this project, I analyzed automobile sales data using Python and Power BI to explore revenue trends, customer behavior, product performance, and geographic sales distribution.

The goal was to extract business insights and build an interactive dashboard to support decision-making.

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

---

## Project Analysis
- Revenue trend analysis
- Product performance analysis
- Customer & RFM analysis
- Customer churn analysis
- Geographic sales analysis

---

## Key Insights
- Q4 generated the highest revenue.
- VIP customers contributed most of the revenue.
- Revenue was concentrated in a limited number of product lines.
- Some high-value customers showed churn risk behavior.
